Easily present your work in progress design project to your client in a private and restricted area.

== Description ==

Projector is a WordPress plugin that lets you showcase a WIP (work in progress) graphic design project to your client.
You'll be able to showcase your work and get better feedback for your visual creations.

== Installation ==

Simply download this plugin, install it to your WordPress site and a new 'WIP' post type will be available.

== Screenshots ==

1. A sample of the new WIP post type with its custom fields.
2. Plugin settings.

== Usage ==

After installation, this plugin will create a 'WIP' new post type and menu item. Simply create new WIP posts, add images and descriptions and protect the page with a password if needed. Then share the post URL with your client.
